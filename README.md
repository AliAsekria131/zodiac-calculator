# نجمك — Najmak

تطبيق ويب لحساب البرج بناءً على اسم الابن واسم أمه.

## بنية المشروع

```
najmak-app/
├── index.html         ← الهيكل فقط، بدون منطق أو styles مدمجة
├── favicon.png        ← أيقونة الموقع
├── css/
│   └── style.css      ← كل التصميم
└── js/
    ├── data.js        ← البيانات الثابتة (قيم الحروف + الأبراج)
    ├── calculator.js  ← المنطق الحسابي البحت (بدون DOM)
    └── ui.js          ← التعامل مع الواجهة فقط
```

## كيف يعمل

1. يُدخل المستخدم اسم الابن واسم الأم بالعربية
2. يُحسب مجموع قيم حروف كل اسم
3. يُجمع الرقمان ثم يُقلَّص بطرح 12 تكراراً حتى الوصول لأصغر قيمة موجبة
4. يُعرض البرج المقابل للرقم الناتج

## التقنيات

- HTML5 + CSS3
- JavaScript (Vanilla)
- Bootstrap 5
- Google Fonts (Tajawal)
